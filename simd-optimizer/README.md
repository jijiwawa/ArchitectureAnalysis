# SIMD Optimizer

一键安装 SIMD 优化开发技能包。

## 安装

```bash
bash install-simd-optimizer.sh
```

## 使用

在 OpenClaw 中说：
- `用 NEON 优化这个循环`
- `帮我做 SIMD 优化`

## 文件

| 文件 | 说明 |
|------|------|
| `install-simd-optimizer.sh` | 一键安装脚本 |
| `simd-optimizer-guide.md` | 详细使用指南 |

## 功能

- 自动识别待优化代码
- 生成 NEON/SVE SIMD 实现
- 生成 GTest 测试用例
- 性能基准对比
